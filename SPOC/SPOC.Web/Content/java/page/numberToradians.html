Java toRadians() 方法

Java Number & Math类

toRadians() 方法用于将角度转换为弧度。

语法

double toRadians(double d)

参数

  • d -- 任何原生数据类型。

返回值

该方法返回 double 值。

实例

public class Test{
    public static void main(String args[]){
        double x = 45.0;
        double y = 30.0;

        System.out.println( Math.toRadians(x) );
        System.out.println( Math.toRadians(y) );
    }
}

编译以上程序，输出结果为：

0.7853981633974483
0.5235987755982988
