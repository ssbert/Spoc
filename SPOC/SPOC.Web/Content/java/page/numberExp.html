Java exp() 方法

Java Number & Math类

exp() 方法用于返回自然数底数e的参数次方。

语法

该方法有以下几种语法格式：

double exp(double d)

参数

  • d -- 任何原生数据类型。

返回值

返回自然数底数e的参数次方。

实例

public class Test{ 
    public static void main(String args[]){
        double x = 11.635;
        double y = 2.76;

        System.out.printf("e 的值为 %.4f%n", Math.E);
        System.out.printf("exp(%.3f) 为 %.3f%n", x, Math.exp(x));  
    }
}

编译以上程序，输出结果为：

e 的值为 2.7183
exp(11.635) 为 112983.831
