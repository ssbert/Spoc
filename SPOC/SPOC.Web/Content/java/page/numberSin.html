Java sin() 方法

Java Number & Math类

sin() 方法用于返回指定double类型参数的正弦值。

语法

double sin(double d)

参数

  • d -- 任何原生数据类型。

返回值

返回指定double类型参数的正弦值。

实例

public class Test{
    public static void main(String args[]){
    
        double degrees = 45.0;
        double radians = Math.toRadians(degrees);

        System.out.format("pi 的值为 %.4f%n", Math.PI);
        System.out.format("%.1f 度的正弦值为 %.4f%n", degrees, Math.sin(radians));

    }
}

编译以上程序，输出结果为：

pi 的值为 3.1416
45.0 度的正弦值为 0.7071
