Java toString() 方法

Java Character类

toString() 方法用于返回一个表示指定 char 值的 String 对象。结果是长度为 1 的字符串，仅由指定的 char 组成。

语法

String toString(char ch)

参数

  • ch -- 要转换的字符。

返回值

返回指定 char 值的字符串表示形式。

实例

public class Test {

    public static void main(String args[]) {
        System.out.println(Character.toString('a'));
        System.out.println(Character.toString('A'));
    }
}

以上程序执行结果为：

a
A
