Java isLowerCase() 方法

Java Character类

isLowerCase() 方法用于判断指定字符是否为小写字母。

语法

boolean isLowerCase(char ch)

参数

  • ch -- 要测试的字符。

返回值

如果字符为小写，则返回 true；否则返回 false。

实例

public class Test {

    public static void main(String args[]) {
        System.out.println( Character.isLowerCase('c'));
        System.out.println( Character.isLowerCase('C'));
    }
}

以上程序执行结果为：

true
false
