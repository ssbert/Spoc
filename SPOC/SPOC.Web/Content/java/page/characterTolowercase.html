Java toLowerCase() 方法

Java Character类

char toLowerCase(char ch)

语法

char toUpperCase(char ch)

参数

  • ch -- 要转换的字符。

返回值

返回转换后字符的小写形式，如果有的话；否则返回字符本身。

实例

public class Test {

    public static void main(String args[]) {
        System.out.println(Character.toLowerCase('a'));
        System.out.println(Character.toLowerCase('A'));
    }
}

以上程序执行结果为：

a
a
