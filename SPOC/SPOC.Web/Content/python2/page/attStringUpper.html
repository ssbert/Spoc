Python upper() 方法

Python 字符串

描述

Python upper() 方法将字符串中的小写字母转为大写字母。

语法

upper()方法语法：

str.upper()

参数

  • NA

返回值

返回小写字母转为大写字母的字符串。

实例

以下实例展示了 upper()函数的使用方法：

#!/usr/bin/python

str = "this is string example....wow!!!";

print "str.upper() : ", str.upper()

以上实例输出结果如下：

str.upper() :  THIS IS STRING EXAMPLE....WOW!!!
