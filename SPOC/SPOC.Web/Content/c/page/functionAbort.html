C 库函数 - abort()

C 标准库 - <stdlib.h>

描述

C 库函数 void abort(void) 中止程序执行，直接从调用的地方跳出。

声明

下面是 abort() 函数的声明。

void abort(void)

参数

  • NA

返回值

该函数不返回任何值。

实例

下面的实例演示了 abort() 函数的用法。

#include <stdio.h>
#include <stdlib.h>

int main ()
{
   FILE *fp;
   
   printf("准备打开 nofile.txt\n");
   fp = fopen( "nofile.txt","r" );
   if(fp == NULL)
   {
      printf("准备终止程序\n");
      abort();
   }
   printf("准备关闭 nofile.txt\n");
   fclose(fp);
   
   return(0);
}
