Python3 字典 keys() 方法

Python3 字典

描述

Python 字典 keys() 方法以列表返回一个字典所有的键。

语法

keys()方法语法：

dict.keys()

参数

  • NA

返回值

返回一个字典所有的键。

实例

以下实例展示了 keys() 方法的使用方法：

#!/usr/bin/python3

dict = {'Name': 'Runoob', 'Age': 7}

print ("字典所有的键为 : %s" %  dict.keys())

以上实例输出结果如下：

字典所有的键为 : dict_keys(['Age', 'Name'])
